What you're seeing

Each shape is an eka tala cycle. Dots (walkers) traverse the vertices — one vertex per beat. When walkers from different cycles land on beat 1 (sam), a convergence flash appears.

Spread mode separates shapes. Overlay mode stacks all shapes on a shared sam vertex — walkers literally meet at the same point when cycles align.
