# Shaping Time: Tala as Spatial and Embodied Experience

## Project Overview

A final project for a course on South Indian musical time. This project explores how **tala** (rhythmic cycles) can be understood not as abstract timekeeping, but as a **relational system that emerges through movement and interaction**.

Participants physically traverse geometric pathways that encode distinct rhythmic cycles. Each step marks a pulse, and when multiple participants walk intersecting paths simultaneously, audible points of synchronization emerge at moments of convergence -- revealing how independent rhythmic cycles coexist within a shared temporal framework.

## Core Argument

Rhythmic time in South Indian music is best understood as **relational and spatial** rather than linear. By translating tala into navigable space and embodied movement, the cyclical and interactive nature of rhythmic structure becomes immediately perceptible.

## Approach

The project is grounded in the **suladi sapta tala system**, using **eka tala** structures across different **jati-s** (rhythmic families):

| Jati | Eka Tala Structure | Beats |
|------|-------------------|-------|
| Chatusra | Clap + 3 finger counts | 4 |
| Tisra | Clap + 2 finger counts | 3 |
| Khanda | Clap + 4 finger counts | 5 |
| Misra | Clap + 6 finger counts | 7 |
| Sankirna | Clap + 8 finger counts | 9 |

Each pathway is designed to reflect the internal logic of its tala, with symmetrical and asymmetrical geometric forms corresponding to different cycle lengths.

### Embodiment Design (To Be Refined)

A key design question: how participants physically express **tala beats** vs. **gati (note-rate/subdivision)**. Options under consideration:

- **Steps = tala beats**, **claps = gati** -- participants step on each beat of the cycle and clap the subdivision pattern
- **Steps = gati pulses**, with a distinct gesture (stamp, voice) for tala beats
- Some other multimodal split

This distinction is critical so that observers and participants can perceive both the cycle structure and the subdivision rate simultaneously.

## Project Stages

1. **Single pathway** -- one participant walks one tala cycle
2. **Layered cycles** -- multiple participants on intersecting paths, generating convergence points
3. **Increasing complexity** -- more cycles, different jati-s, demonstrating polyrhythmic interaction

## Key Concepts

- **Tala** -- rhythmic cycle defined by a pattern of claps and finger counts
- **Jati** -- rhythmic family that determines the number of beats in a tala component (Chatusra=4, Tisra=3, Khanda=5, Misra=7, Sankirna=9)
- **Gati** -- note-rate or subdivision within a beat
- **Eka Tala** -- a tala consisting of a single clap + finger counts
- **Suladi Sapta Tala** -- the system of seven tala structures, each of which can take any of the five jati-s
